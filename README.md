# Library Information Management System

## Description
The Library Information Management System is a web-based application designed to manage library resources such as books, users, borrowing, and returns efficiently.

## Features
- Book management (add, update, delete)
- User registration and management
- Borrowing and return tracking
- Admin dashboard
- Search functionality

## Technologies Used
- PHP
- MySQL / SQLite
- HTML, CSS, JavaScript

## Installation
1. Clone the repository
2. Import the database file
3. Configure the database connection
4. Run the system on localhost (XAMPP)

## Usage
Admins manage books and users, while users can search and borrow available books.
